# CareerPath Docker Setup

## Quick Commands
- Start Dev: `.\scripts\start-env.ps1 dev`
- Start Prod: `.\scripts\start-env.ps1 prod`
- Stop All: `.\scripts\stop-env.ps1`

- Frontend: http://localhost:4200
